# Ecommerce App

## Features
- User Registration & Login
- Role-Based Access Control (Admin/User)
- Product Management
- User Profile Management
- Google SSO Login

## Tech Stack
- Spring Boot (Backend)
- React (Frontend)
- MySQL (Database)

## Roles
- Admin → Can add/update/delete products
- User → Can only view products

## Run Project

### Backend
Run Spring Boot Application

### Frontend
npm install
npm start
